# Privacy Policy — 7BTVFZ Emote Picker

**Last updated: March 17, 2026**

## Data Collection
7BTVFZ Emote Picker does not collect, store, or transmit any personal user data to external servers.

## Permissions Used
- **activeTab / tabs / webNavigation** — used solely to detect when the user is on twitch.tv and inject the emote picker UI.
- **storage** — used to save your favorite emotes and picker settings **locally in your browser only**. This data never leaves your device.
- **host_permissions (twitch.tv, 7tv.io, api.betterttv.net, api.frankerfacez.com)** — used to fetch publicly available emote data (names, images) from third-party emote providers. No personal data is sent.

## Third-Party Services
The extension fetches emote data from:
- [7TV](https://7tv.app) — [Privacy Policy](https://7tv.app/privacy)
- [BetterTTV](https://betterttv.com) — [Privacy Policy](https://betterttv.com/privacy)
- [FrankerFaceZ](https://www.frankerfacez.com) — [Privacy Policy](https://www.frankerfacez.com/privacy-policy)

The extension does **not** send any data to these services beyond what is required to load emote images.
